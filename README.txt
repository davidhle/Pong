 ------------
|INSTRUCTIONS|
 ------------
 -Use the up and down arrow keys to move the first/cyan player
 -Use the left and right arrow keys to move the second/green player
 -If a player scores a point, then there will be a square with the player's respective color to represent a point
 -If a player reaches 3 points, the player wins, the last point will be red, and the game will end
